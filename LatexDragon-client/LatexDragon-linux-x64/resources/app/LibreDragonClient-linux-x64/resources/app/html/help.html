Timeline
La timeline contient tous les états de la partie, du début à l'état actuel, l'état réprésenté en rouge est l'état actuel, ceux à gauche de l'état rouge sont les précédent, ceux à droite sont les suivants. Cliquer sur l'un de ces états va vous placer sur l'état sur lequel vous avez cliquer. Si vous n'êtes pas sur le dernier état dans la liste et que vous effectuer une action (appliquer une règles) tout les états suivant seront perdu.

- Ces boutons permettent d'afficher/cacher la timeline.

- Ce bouton vous permet de passer en mode 'théorème' dans ce mode cliquer sur les éléments de la timeline les sélectionnera pour la création d'un théorème.

- Ce bouton vous permet de valider la création d'un théorème, pour créer un théorème vous devez sélectionner 2 éléments dans la timeline, une fois sléctionner cliquer sur ce bouton fera apparaître une popup vous demandant la confirmation, une fois confirmer le théorème sera créer.

- Ce bouton vous permet d'annuler la création d'un théorème de revenir en mode 'normal'.

Fonctionnement du jeu
Si vous êtes en mode NORMAL vous avez un certain temps pour finir la partie. Si vous êtes en mode SANDBOX vous n'avez aucune limitation (il n'y a pas de victoire).
Pour apliquer une règle à la formule faite clique droit sur une partie de la formule, un menu va apparaître vous affichant toutes les règles qui sont applicable à l'élément sélectionner. Cliquer (clique gauche) sur l'une des ces règles pour l'appliquer à la formule. Si vous voulez fermer ce menu fait clique droit ou gauche ailleur sur l'écran.
